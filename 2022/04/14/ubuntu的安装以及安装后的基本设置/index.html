vim编辑器如何删除一行或者多行内容 请参考：

我们这里是删除全部，直接：

  • 按一下ESC键，确保退出编辑模式
  • 按一下:冒号键，(shift + ;)就可以输入：冒号了。
  • 然后输入1,$d， 回车就可以删除全部内容了。

vim 设置行号：

1 为所有用户设置行号

直接更改配置文件：sudo vim /etc/vimrc，在最后一行加上 set nu 就可以啦。如果没有改文件那就新建。

2 为当前用户设置行号（※※）

直接更改配置文件：vim ~/.vimrc，在最后一行加上 set nu 就可以啦。如果没有改文件那就新建。
