第一章 el表达式

全称:Expression Language(表达式语言) jsp2.0以后内置el

1 作用:

​ 主要用来代替 <%= %>

2 格式:

​ ${表达式}

3 常见功能

获取作用域中的数据 ★

==便捷查找==

​ ${域中属性的名称}

注意事项

​ 属性名中一旦使用了”.” “+” “-“等特殊符号的时候${属性名}就不起作用了,要使用{域Scope[“属性名”]}
​ 例如:${requestScope[“aa.bb”]}

示例代码:

获取复杂数据

1 获取数组中的数据

​ ${数组属性名称[index]}

​ 例子: request.setAttribute(“arr”, new String[]{“aa”,”bb”,”cc”});
​ 获取的时候：${arr[1]}

2 获取list中的数据

​ 格式：${list属性名[index]}

3 获取map中的数据

​ ${map属性名称.键名}

4 获取javabean中的数据

​ ${javabean在域中的属性名称.属性名称}

执行简单运算

算术运算符

​ + - * / %

关系运算符

返回结果为 Boolean值

​ > 或者 gt， 例如：${8>9} 或者 ${8 gt 9 }

​ >= 或者 ge， 例如：${45>=9} 或者 ${45 ge 9 }

​ < 或者 lt， 例如：${4<9} 或者 ${4 lt 9 }

​ <= 或者 le， 例如：${9<=8} 或者 ${9 le 8 }

​ == 或者 eq， 例如：${4==4} 或者 ${4 eq 4 }

​ != 或者 ne， 例如：${4!=3} 或者 ${4 ne 3 }

逻辑运算符
三元运算符

​ ? :
​ 例如：${3>2?’是’:’不是’}

==特殊运算符==

​ empty
​ 判断EL表达式中的表达式是否为空,返回结果为Boolean值

第二章 jstl

​ apache出品的标签库语言

作用:

​ 增强jsp页面的功能, 在jsp页面上少写java代码

分类

​ core:核心(目前还在使用)

​ fmt:格式化(国际化)

​ xml:和xml相关的(过时了) json

​ sql:和sql相关的(过时了)

​ fn:对字符串处理的函数库(用的很少了)

使用步骤:

​ 1.导入jar包

​ 2.在页面上引入标签库
​ <%@ taglib prefix=”c” uri=”http://java.sun.com/jsp/jstl/core" %>
